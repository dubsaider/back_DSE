# back_DSE
backend of the web interface of the smart video surveillance system

## Environment Variables

To run this project, you will need to add the following environment variables to your `.env` file:

```env
DJANGO_SECRET_KEY=
DJANGO_PORT=
DEBUG=
KAFKA=
RTSP_SERVER=
BACKEND=
POSTGRES_DB=
POSTGRES_USER=
POSTGRES_PASSWORD=
POSTGRES_HOST=
POSTGRES_PORT=
K8S_ADDRESS=
```
